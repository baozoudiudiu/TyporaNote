清除浮动的四种方法

![A714CDF4-B2DE-4D39-A340-698A30C78179](清除浮动.assets/A714CDF4-B2DE-4D39-A340-698A30C78179.png)

```
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>清除浮动</title>
    <style type="text/css">
        .one {
            width: 400px;
            background-color: pink;
        }
        .one .div1 {
            width: 100px;
            height: 30px;
            float: left;
            background-color: red;
        }
        .one .div2 {
            width: 100px;
            height: 50px;
            float: left;
            background-color: orange;
        }
        .clear {
            clear: both;
        }


        .two {
            width: 400px;
            background-color: blue;
            margin: 10px 0 0px;
            overflow: hidden;
        }
        .two .div1 {
            float: right;
            height: 60px;
            background-color: yellow;
        }




        .three {
            width: 400px;
            background-color: cyan;
            margin: 10px 0 0;
        }
        .three .div1 {
            float: left;
            height: 300px;
            background-color: gold;
            width: 100px;
        }
        .clearfix:after {
            content: "";
            display: block;
            height: 0;
            visibility: hidden;
            clear: both;
        }
        .clearfix {
            *zoom:1;
            /*ie6,7专门清除浮动的样式*/
        }

        /*
        方法四
        .clearfix:before,
        .clearfix:after {
            content: "";
            display: table;
        }
        .clearfix:after {
            clear: both;
        }
        .clearfix {
            *zoom:1;
        }
         */
    </style>
    </style>
</head>
<body>
    <!-- 方法一 -->
    <div class="one">
        <div class="div1">11111</div>
        <div class="div2">22222</div>
        <div class="clear"></div>
    </div>
    <!-- 方法二 -->
    <div class="two">
        <div class="div1">11111</div>
    </div>
    <!-- 方法三 -->
    <div class="three clearfix">
        <div class="div1">333333</div>
    </div>
</body>
</html>
```
